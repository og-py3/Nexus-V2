## Features
NEXUS V2 CHECKER - FEATURES

⚡ CORE ENGINE
- Multi-threaded checking system for high speed performance
- Bulk input support (load large combo/proxy lists from files)
- Auto input cleaner (removes duplicates, invalid formats, empty lines)
- Live statistics panel (Hits / Fails / Retries / Checked / Speed)
- Auto-save results (hits.txt, bad.txt, retries.txt)

🌐 PROXY CHECKER MODULE
- Supports HTTP / HTTPS / SOCKS4 / SOCKS5 proxies
- Proxy latency testing (response time in ms)
- Dead proxy detection and filtering
- Auto retry on timeout or failure
- Optional geo-info lookup (country / ISP via API)

🔐 CHECKER SYSTEM (GENERIC)
- Modular checker support (plug-and-play structure)
- API-based validation system (custom endpoints support)
- Status classification system (valid / invalid / retry)
- Smart retry logic with limits to avoid overload
- Rate-limit handling with adaptive delay system

🎛️ UI / TERMINAL EXPERIENCE
- Clean terminal dashboard UI
- Color-coded output (green = hits, red = fails, yellow = retries)
- Real-time progress updates
- Command controls (start / stop / pause / resume / reset)
- Config-based customization (no hardcoding needed)

🛡️ STABILITY & PERFORMANCE
- Fully error-handled (no crash on bad inputs)
- Thread limiter for stable performance
- Timeout control for requests
- Auto resume from last checkpoint
- Input validation before processing starts

📦 ADVANCED FEATURES (PRO LEVEL)
- Modular plugin system (add/remove checker modules easily)
- JSON / CSV export for results
- Config loader (editable settings file)
- Optional remote config updater
- Structured project architecture for scalability
